# 📘 CBSE Class 10 Mathematics Blueprint (2026–27)

A clean, structured, and student-friendly web interface for the **CBSE Class 10 Mathematics Curriculum (2026–27)**.
Built to help students organize their study, track progress, and focus on high-weightage topics.

## 🎯 Purpose

This project is designed to:

* Simplify the CBSE curriculum into a clear blueprint
* Help Class 10 students plan their preparation effectively
* Provide a distraction-free study interface
* Lay the foundation for a full study tracking system
---

## ✨ Features

* 📚 Chapter-wise structured layout
* 🎯 Focus on important topics & weightage
* 💻 Clean and minimal UI
* 📱 Responsive design (mobile, tablet, desktop)
* ⚡ Fast and lightweight (pure HTML + CSS)
---
